# X (formerly Twitter) Clone

A feature-rich Twitter clone built with React and Vite.

## Recently Added Features (Apr 18 - Apr 25, 2026)

- **Comment/Reply System**: Nested replies with real-time UI updates.
- **Repost/Retweet System**: Ability to repost tweets with a "Reposted" indicator.
- **Like Enhancement**: Heart animation and toggle (unlike) support.
- **Loading Skeletons**: Smooth skeleton loading for the main feed.
- **Character Counter**: 280-character limit with real-time counter and validation.
- **Time Ago Format**: Relative timestamps (e.g., 5m, 2h) for posts.
- **Delete Confirmation**: Secure deletion with a modal confirmation.

## Tech Stack
- React
- Vite
- LocalStorage for persistence
- Vanilla CSS for styling
